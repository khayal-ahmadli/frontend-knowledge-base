# ✅ Flexbox vs Grid — Əsas Fərqlər

---

## ✅ 1. Ölçü sayı (1D vs 2D)

Bu ən böyük və fundamental fərqdir.

- **Flexbox → 1D layout sistemidir**  
  Ya row, ya column istiqamətində işləyir.

- **Grid → 2D layout sistemidir**  
  Row + column eyni anda idarə oluna bilir.

**Qısa:**  
- Flex → 1 ölçü  
- Grid → 2 ölçü

---

## ✅ 2. İstiqamət

- Flexbox-da elementlər **bir xətt boyunca düzülür**.  
- Grid-də elementlər **şəbəkə (grid) şəklində sətir-sütun üzrə** düzülür.

---

## ✅ 3. Layout-un mürəkkəbliyi

**Flex — sadə layout-lar üçün idealdır:**
- navbar  
- card düzülüşü  
- sidebar + content  

**Grid — daha kompleks layout-lar üçündür:**
- dashboard  
- admin panel  
- gallery  
- çox bölməli dizaynlar  

---

## ✅ 4. Space bölüşdürmə (paylanma)

- Flex → content-based  
  (elementlərin daxilindəki məzmun ölçü yaradır)

- Grid → layout-based  
  (əvvəlcə grid xətləri qurulur, sonra content yerləşir)

---

## ✅ 5. Responsivlik

- Flex responsivdir, amma bəzi dizaynlarda sətirlər qarışa bilər.  
- Grid-də `minmax`, `fr`, `auto-fit`, `auto-fill` → **responsiv dizaynı mükəmməl edir**.

---

## ✅ 6. Sətir və sütunları idarəetmə

- Flex → yalnız **bir xətt** idarə edir  
- Grid → həm **sətir**, həm **sütun** ayrıca idarə olunur
