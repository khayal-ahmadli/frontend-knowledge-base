# Inline vs Block Elementlər və Box Model Davranışı

## ✅ Block-level elementlər

**Xüsusiyyətləri:**
- `width: 100%` (default olaraq tam sətiri tutur)
- `margin` və `padding` həm vertical, həm horizontal işləyir
- Həmişə **yeni sətirdə** başlayır

**Nümunələr:**
- `<h1> ... <h6>`  
- `<p>`  
- `<ul>`, `<ol>`, `<li>`  
- `<form>`  
- `<table>`  
- `<div>`  
- `<section>`  
- `<header>`  
- `<footer>`  

---

## ✅ Inline elementlər

**Xüsusiyyətləri:**
- `width` və `height` verilmir (əcza ignore olunur)
- `margin-top` və `margin-bottom` işləmir  
- `padding` həm horizontal, həm də vertical işləyə bilər, amma vertical layout-u pozmur
- Eyni sətirdə davam edir, yeni sətir açmır

**Nümunələr:**
- `<span>`  
- `<a>`  
- `<b>`  
- `<i>`  

---
